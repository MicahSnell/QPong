# QPong
Classic pong made with Qt5.

## Build and Run
```bash
$ mkdir build && cd build
$ qmake-qt5 ..
$ make
$ ./qpong
```

## Documentation
Documentation can be generated with `doxygen`
```bash
$ cd doc
$ doxygen Doxyfile
$ <web browser> ./html/index.html
```
